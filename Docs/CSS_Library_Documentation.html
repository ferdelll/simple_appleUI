CSS Library Documentation

This library provides ready-to-use CSS styles inspired by macOS. It includes styles for buttons, input fields, text elements, links, and context menus.

1. Buttons

Styled buttons with hover and active effects:

<button class="mac-button">Default Button</button>
<button class="mac-button mac-button-accent">Accent Button</button>
<button class="mac-button mac-button-disabled" disabled>Disabled Button</button>

2. Input Fields

Styled input elements including text, checkboxes, and radio buttons:

<input type="text" class="mac-input" placeholder="Enter text">
<input type="checkbox" class="mac-checkbox">
<input type="radio" class="mac-radio">

3. Sliders

A modern slider design:

<input type="range" class="mac-slider" min="0" max="100">

4. Text Elements

Styled text elements for various use cases:

<p class="mac-text">Default Text</p>
<p class="mac-text-small">Small Text</p>
<p class="mac-text-large">Large Text</p>
<p class="mac-text-muted">Muted Text</p>

5. Links

Styled links with hover and active effects:

<a href="#" class="mac-link">Styled Link</a>

6. Context Menu

A styled context menu with interactivity:

<ul class="mac-context-menu">
    <li class="mac-context-menu-item">Option 1</li>
    <li class="mac-context-menu-item">Option 2</li>
    <li class="mac-context-menu-item">Option 3</li>
</ul>

Add the following JavaScript for functionality:

document.addEventListener('contextmenu', (e) => {
    e.preventDefault();
    const menu = document.getElementById('context-menu');
    menu.style.left = `${e.pageX}px`;
    menu.style.top = `${e.pageY}px`;
    menu.classList.add('active');
});

document.addEventListener('click', () => {
    const menu = document.getElementById('context-menu');
    menu.classList.remove('active');
});
